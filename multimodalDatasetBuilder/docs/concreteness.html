concreteness module¶

class concreteness.Concreteness(path_to_concreteness_values_file: str, concreteness_threshold: float)¶

Bases: object

This class is responsible for finding concrete/depictable words. For that, a file containing the concreteness values of words is used.

Parameters:

  • path_to_concreteness_values_file (str) – path to the json file containing the concreteness values

  • concreteness_threshold (float) – a value to filter out the words from the concreteness file which have a value below the specified threshold
