runminiclip module¶

class runminiclip.Runminiclip(path_to_highlighted_imgs: str, path_to_resized_main_imgs: Optional[str] = None)¶

Bases: object

The class that uses miniCLIP to highlight images. If specified the main images of the sentences can be resized to match the size of the highlighted images from miniCLIP.

Parameters:

  • path_to_highlighted_imgs (str) – path where the highlighted images will be saved

  • path_to_resized_main_imgs (str) – path where the resized version of the main images will be saved. If not specified the main images for the multimodal sentences will keep their original size.
