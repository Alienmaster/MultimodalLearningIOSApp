duplicateWordsRemover module¶

duplicateWordsRemover.remove_duplicate_words(words: []) → []¶

Removes duplicate words from the list if these occur similarly in different forms again (e.g., “apple” and “APPLES” are duplicates in this context).

Parameters:

words ([]) – list of words (nouns) to be processed

Returns:

list of words (nouns) without duplicates

Return type:

[]
